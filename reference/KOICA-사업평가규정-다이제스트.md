## 제1조 목적
평가를 통한 **지식·교훈 학습과 기관 책무성 향상**.

## 제5조 평가원칙 (5)
1. **파트너십** — 수원·공여기관 참여·공동평가 장려
2. **공정성(Impartiality)** — 균형적·편견 없는 평가
3. **객관성(Objectivity)** — 가치판단과 사실 구분
4. **투명성(Transparency)** — 목적·방법론 명기, **사실·결론과 가치판단·제언을 구분**
5. **진실성(Credibility)** — 신의성실, 투명한 과정

## 제6조 평가기준 (7) — 우리 6기준의 규정적 근거
| 규정 기준 | 우리 시스템 |
|----------|------------|
| 적절성 Relevance | `dac-relevance-evaluator` |
| 효과성 Effectiveness (취약계층 포용·형평성 포함) | `dac-effectiveness-evaluator` |
| 효율성 Efficiency | `dac-efficiency-evaluator` |
| **영향력 Impact** (장기·전환적 효과) | (종료평가 종합점수 제외, 사후/영향평가에서) |
| 지속가능성 Sustainability | `dac-sustainability-evaluator` |
| 일관성 Coherence | `dac-coherence-evaluator` |
| 범분야 이슈 (젠더·환경·인권) | (효과성 형평성 등에 반영) |
- ②: **사업별 특성 고려해 평가기준을 선택적으로 적용 가능.** (CTS 사업의 타당성 추가도 이 재량 범위.)

## 제7조 평가유형 — 우리 두 모드의 근거
**시기별:** 사전평가 / 중간평가 / **종료평가**(종료시점 성과목표 달성도) / **사후평가**(종료 후 장기성과 도출·유지).
- **종료평가** = 우리 6기준 사업평가팀 (DAC 6기준 → A~F).
- **영향평가** = 별도 가이드라인(평가실 주관)상 인과효과 측정 평가 → `impact-evaluation-reviewer`. (규정 제7조 평가유형 체계 + 별표3 평가성 사정과 연계.)
**평가자별:** 외부 / 내부 / 공동.

## 제19조 평가독립성 보장
평가자의 독립성·자율성을 보장한다. 즉, 어느 한쪽이 평가 내용을 임의로 고치거나 빼거나 덧붙이도록 강요하는 등 부당한 요구를 하지 못하게 한다.
→ 우리 "사람 게이트"와 정합: AI는 초안, 사람이 확정하되 **평가 독립성**을 침해하지 않는다.

## 제23조 평가대상 등급 부여
국내·현지조사 후 성과·한계를 체계 분석해 **평가등급 부여**(종료평가 업무수행 길라잡이 준용).
→ 우리 A~F 등급 산정의 근거.

## 제27·28조 평가 품질검토 — 우리 품질심사관의 근거
- **평가품질검토위원회**: 외부 전문가, **독립평가패널 풀**, 패널 **3인** 구성. 품질검토 대상과 이해관계 있는 위원 제외.
- **별표1 품질검토 기준**으로 **A·B·C·D 등급** 부여, 위원장이 3인 결과 종합 → 최종품질등급·통과여부 판정·의결.
- **블라인드 검토**(평가자 정보 비공개) 원칙.
→ `report-quality-inspector`가 정확히 이 구조(24문항/A~D/블라인드/패널평균).

## 제6장 평가윤리 (제39~48조)
윤리준수(제39) / **윤리원칙: 독립성** 등(제40) / 평가참여자 배려(제41) / **익명성 보장**(제42) / 부정적 영향 최소화(제43) / **비밀엄수·개인정보 보호**(제46) / 성희롱·성폭력 예방(제48).
→ 우리 에이전트 공통 규칙(익명 처리·개인정보·독립성).

## 별표3 평가성 사정 기준·체크리스트 (2024.07.30 신설)
**영향평가 대상 적합성** 사정 → `impact-evaluation-reviewer`의 대상 적합성 사전심사와 연계.
